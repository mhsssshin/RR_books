# 📚 로롱북스 (rorongBooks) 2차 개선 완료 보고서 (Walkthrough)

동화책 분량 확장(10페이지 내외), 어절(단어) 단위 선택형 TTS 구현, 그리고 대표 책 2권의 10페이지 삽화 페어링 및 Git 원격 업로드를 완벽히 끝마쳤습니다!

추가로 두 번째 창작 동화인 **"꼬마 토끼의 용기 상자"** 역시 1페이지당 1개의 고유 일러스트를 탑재할 수 있도록 구조 개편 및 파일 연동을 완료했습니다.

---

## 🛠️ 기능 개선 사항 상세

### 1. 단어 단위 선택형 TTS (Word-by-Word TTS)
- **개선 사유**: 아이가 한 페이지 전체 문장을 들으면 집중력이 떨어질 수 있어, 모르는 단어나 흥미로운 단어만 꾹 눌러 개별적으로 들어볼 수 있도록 기능을 고도화했습니다.
- **작동 원리**:
  - [books/*.html](file:///c:/Project/rorongBooks/books) 내의 원본 동화 텍스트는 컴파일 시에 일반 텍스트 문단으로 빌드됩니다.
  - 책이 실행되면 [js/book.js](file:///c:/Project/rorongBooks/js/book.js)가 문단을 공백(`\s+`)을 기준으로 쪼개어 각 단어를 자동으로 `<span class="tts-trigger">단어</span>` 태그로 감쌉니다.
  - 아이가 원하는 단어를 꾹 누르면 그 낱말 주변으로만 무지갯빛 진행률 바가 차오르고, 3초가 지나면 **해당 단어 하나만** Web Speech API로 정확하고 다정하게 읽어줍니다. 

### 2. 동화 분량 전면 개편 (10페이지 고정)
- [data/stories.json](file:///c:/Project/rorongBooks/data/stories.json) 내의 모든 30개 동화에 대해 4페이지 구조에서 **10페이지 분량**의 정교한 스토리라인으로 원고를 확장 및 수정하였습니다.
- 난이도를 5세 여아 수준에 맞춰 1~2개 어절 문장으로 짤막하게 구성하여 글을 읽는 훈련을 돕습니다.
- 이에 맞춰 10페이지 완독 직후에만 별점 모달창이 표시되도록 트래커를 자동 조정했습니다.

### 3. 대표 도서 전면 삽화 매핑 및 이미지 생성
- **창작 도서 대표 1**: `creative_1` (로롱이와 무지개 요정) - 표지 및 1~10페이지 본문용 고화질 일러스트 5장 생성 및 페어링 완료 (2페이지당 1장)
- **창작 도서 대표 2 (추가)**: `creative_2` (꼬마 토끼의 용기 상자) - 표지 및 1~10페이지 본문용 1페이지당 1개의 개별 일러스트 구조 세팅 완료 (총 10장)
- **명작 도서 대표 1**: `classic_1` (신데렐라) - 5장 고화질 일러스트 생성 및 페어링 완료 (2페이지당 1장)

> [!NOTE]
> **2번째 책 일러스트 가이드 (JPG 최적화)**
> - 사용자께서 직접 전달해 주신 고품질 `.jpg` 파일들(`creative_2_1.jpg` ~ `creative_2_10.jpg`)을 각 페이지에 1장씩 정교하게 매핑 완료하였습니다.
> - 웹 로딩 속도 최적화를 위해 JPG 압축 규격을 정상적으로 적용하여 코드 컴파일을 마쳤습니다.

## 📂 파일 구조 및 컴파일 실행
- 모든 변경된 HTML 코드는 `npm run compile`로 재생성되어 [books/](file:///c:/Project/rorongBooks/books) 아래 30권 전체가 업데이트되었습니다.
- 새로 추가된 일러스트들은 [assets/images/](file:///c:/Project/rorongBooks/assets/images/)에 담겨 있으며, 깨진 링크가 발생하지 않도록 대체 렌더링 및 페어링 처리가 완벽히 적용되었습니다.
- 모든 수정 코드가 원격 저장소(`main` 브랜치)로 최종 업로드(Push) 완료되었습니다.

## 🚦 검증 가이드
1. 배포된 주소 `https://mhsssshin.github.io/RR_books/` 에 접속합니다.
2. `로롱 창작 동화` 탭에서 **꼬마 토끼의 용기 상자**를 선택합니다.
3. 1~10페이지까지 넘어가며, 1페이지당 1개의 삽화(현재 토끼 삽화로 페어링)가 정상적으로 표시되는지 확인합니다.
4. 본문 단어(예: "토토는", "어두워져서", "황금") 중 하나를 3초간 꾹 눌렀을 때 해당 단어가 예쁘게 하이라이트되면서 **오직 그 낱말만 발음해 주는지** 검증합니다.
5. 페이지를 10페이지까지 전부 넘긴 직후 우아하게 날아오르는 별점창 및 제출 성공 시의 꽃가루(Confetti) 폭죽이 정상 동작하는지 테스트합니다.
